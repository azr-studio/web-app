# 🚀 小安题库 - AI 备考大师 (至臻版)

[![GitHub Pages](https://img.shields.io/badge/Live-Demo-brightgreen)](https://azr-studio.github.io/web-app/)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

这是一个基于原生 Web 技术开发的**全功能备考系统**，专为高效复习 SSM 框架和微机原理设计。

## ✨ 核心功能
- 📖 **多种模式**：支持“背题模式”查看详解与“练习模式”实时纠错。
- 📝 **模拟考试**：随机抽取题目，倒计时限时模拟真实考试环境。
- 📊 **智能统计**：实时生成考试分析报告，通过 Chart.js 展示正确率分布。
- ⚠️ **错题追踪**：自动记录错题，支持针对性强化复习。
- 🌓 **个性化体验**：支持明暗主题切换，多端适配（移动端友好）。

## 📚 覆盖科目
- **SSM 框架**：包含 MyBatis、Spring AOP、Spring MVC、事务管理等核心考点。
- **微机原理**：涵盖 8086 CPU、中断系统、8259A/8253/8255A 接口芯片等。

## 🛠️ 技术栈
- **前端**：HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6)
- **可视化**：[Chart.js](https://www.chartjs.org/)
- **部署**：GitHub Pages

## 📖 如何使用
1. 直接访问 [在线地址](https://azr-studio.github.io/web-app/)。
2. 选择上方科目开始练习。
3. 点击“📊 进度”可以快速跳转题目。
